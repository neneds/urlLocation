# urllocation

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)


urllocation provides an parse for url's generated by location apps such as Google Maps, Apple maps and here maps. And it returns a JSON with the coordinates from the url. 

# URL'S supported

  - goo.gl - Google maps URL
  - maps.google - Google maps URL
  - maps.apple.com - Apple maps URL
  - her.is - Here maps URL
  - wego.here.com - Here maps URL
  - 
### Installation

Install the dependencies 

```sh

$ npm install @neneds/urllocation
```
